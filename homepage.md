# 個人網頁 Homepage

## 個人網頁 Homepage

以帳號為 b99902067 為例：（自行設定時請換成自己的帳號）

###設定 
在個人目錄下進入 (或建立) htdocs 目錄。
```shell
mkdir ~/htdocs
cd ~/htdocs
```
htdocs 目錄下的 index.html 為系統預設之讀取檔案。

###網頁位址 URL
在本系首頁後接 ~USERNAME 即可。
